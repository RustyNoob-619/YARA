# Collection of my YARA rules

## Please see the below:

=> Watch out for YARA rule files starting with the VT naming convention as these will not work outside Virus Total. There are also rules beginning with SUS_ which are intended to highlight suspicious files and not malicious.

=> These rules are written from Threat Research perspective and hence performance considerations are not taken into account.

=> Watch out for the YARA rules starting with the TTP naming convention as these are generic in nature and do not on their own mean malicious but should rather me used in combination with other rules

=> If you spot any rules that are incorrect, please use the issues feature to let me know. For instance rules with incorrect attribution to malware/actors, rules that do not run, hitting on FPs, inaccurate metadata or something else. Efforts have been made to QA the rules but mistakes can be made.
